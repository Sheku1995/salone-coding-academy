About Us

Salone Coding Academy is a grassroots initiative dedicated to bridging the digital skills gap in Sierra Leone. It’s a one-person effort aimed at making tech education affordable and accessible to everyone — especially those who cannot afford university or expensive private institutions.

This bootcamp offers beginner-friendly coding lessons in HTML, CSS, JavaScript, and more — all taught in Krio to make learning easier and more relatable. The main focus is on hands-on training and practical support to help learners gain the skills they need to break into the tech industry.

The mission is simple: to empower individuals with valuable web development skills and open doors to new opportunities.

What you're going to learn

We offer a variety of courses designed to equip you with the skills needed to thrive in the tech industry. Our curriculum is tailored to meet the needs of beginners and those looking to enhance their coding skills.

HTML & It’s Basics

Learn the fundamentals of HTML, the backbone of web development. This course covers everything from basic tags to creating structured web pages.

CSS & It’s Basics

Dive into CSS and learn how to style your web pages. This course covers everything from basic styling to advanced layout techniques.

JavaScript

Master the basics of JavaScript, the programming language of the web. This course covers everything from variables to functions and DOM manipulation.

React

Learn how to build dynamic web applications using React. This course covers everything from components to state management and routing.

Bootstrap

Discover how to create responsive and mobile-first web pages using Bootstrap. This course covers everything from grid systems to components and utilities.

Git & Github

Learn how to use Git and GitHub for version control and collaboration. This course covers everything from basic commands to branching and merging.

By the end of the day, you will be able to:

Understand the fundamentals of web development

Gain a solid understanding of HTML, CSS, and JavaScript to create visually appealing and functional websites.

Build a portfolio of projects

Create a portfolio showcasing your skills and projects. This will help you stand out to potential employers and demonstrate your coding abilities.

Join a supportive community

Become part of a vibrant community of learners and mentors. Share your experiences, ask questions, and collaborate with others to enhance your learning journey.

How is it going to be done

Virtually

Attend classes from the comfort of your home. Our online platform allows you to learn at your own pace and convenience.

One-to-one

Get personalized guidance and support tailored to your learning needs. One-on-one sessions ensure you grasp every concept effectively.

Group

Collaborate with fellow learners in group sessions. Share ideas, work on projects together, and learn from each other’s experiences.
